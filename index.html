About Me

I have embarked on a self-learning journey, using resources like tutorials and platforms such as freeCodeCamp. Through this process, I've built various projects that continuously expand my knowledge and skills as a software developer.

Microsoft Rewards Point Automater

Developed a Java-based application to automate Microsoft Reward tasks which includes a user-friendly GUI for ease of use.


It functions by generating search results for 30 randomly selected words, by reading a text file of every word in the English Language, in Microsoft Edge. This then allows users to earn 90 points per day through automated searches which saves time and effort.

Luxury Camp Site Management System

Used Java with JavaFX to build a multi-featured application for a degree-level assignment which is nearly 800 lines of code long.


Allows the user (i.e. the owner) to switch between four different areas of the site, check customers in/out and update a table to show which rooms are occupied, available and need cleaning. It also validates rooms to meet certain requirements before customers can be checked into them.

Call of Duty Camo Tracker

I am currently engrossed in the development of a website utilising web development technologies (HTML, CSS, JavaScript etc.) which seeks to establish a user-friendly solution for tracking the overall completion of Call of Duty camos.


The inspiration behind this website stems from the notoriously complex and cluttered UI of the game itself which thus causes the task of monitoring camo completion status to be even more complicated. For context, camos in Call of Duty are awarded when a player completes a challenge with a specific weapon e.g. to get 50 headshots with that weapon.
